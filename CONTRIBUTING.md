# Contributing to Trip Planner

Thank you for your interest in the Trip Planner project! This repository is dedicated to issue tracking and community discussions.

This repository **only accepts issues** (bug reports, feature requests, and discussions). It does not accept pull requests with code changes.

## How to Report Issues

### Before Creating an Issue

1. **Search existing issues**: Check if your issue has already been reported
2. **Check the new repository**: Active development happens in [new-tripplanner-ui](https://github.com/lukepower/new-tripplanner-ui)

### Creating a New Issue

1. Click the "New Issue" button
2. Choose the appropriate issue template
3. Fill out the template completely
4. Add relevant labels
5. Submit your issue

### Issue Types

- **Bug Report**: Something isn't working as expected
- **Feature Request**: Suggest a new feature or enhancement
- **Question**: Ask questions about the Trip Planner
- **Discussion**: Start a discussion about the project direction

## Code Contributions

**Code contributions should be made to [new-tripplanner-ui](https://github.com/lukepower/new-tripplanner-ui)**, not this repository.

This repository serves as:
- A central issue tracker
- A reference for historical context
- A place for community discussions

## Questions?

If you have questions about contributing, please open an issue with the "Question" label.
